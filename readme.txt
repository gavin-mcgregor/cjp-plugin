== Installation ==

Adding new blocks to this plugin.

1. Terminal > cd src
2. npx @wordpress/create-block --no-plugin
3. You can accept the warning about system requirements
4. chose: 
    — dynamic block
    — name
    — cjp-blocks
    — name
    — blank
    — blank
    — widgets (change later)
5. cjp-blocks.php > add "register block" to list
